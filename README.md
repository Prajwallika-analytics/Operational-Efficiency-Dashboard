# Operational Efficiency & Cost Leakage Analysis

## Business Problem
A company faces rising operational costs across regions.
Leadership needs to identify where money is leaking,
which regions are over budget, and what's driving the gap.

## What I Built
A full end-to-end Power BI dashboard analyzing 800 operational
records across 5 regions — from raw data to executive insights.

## Key Findings
- 🔴 North region: 18% over budget → $796K annual cost leakage
- 📉 Process delays above 5 days correlate with 12% cost increase
- 💡 10% efficiency improvement could recover ~$796K annually

## Tools Used
- Python (data generation — Pandas, NumPy)
- Power BI (dashboard, KPIs, variance analysis)

## Files in This Repository
- `ops_data.csv` — the dataset (800 rows, 5 regions)
- `dataset_generator.py` — Python script that built the data
- `dashboard_screenshot.pdf` — the final Power BI dashboard
